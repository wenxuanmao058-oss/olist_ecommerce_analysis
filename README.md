# 🛒 Olist 巴西电商数据分析

> 基于 Kaggle 公开数据集，对 Olist 巴西电商平台进行全面的业务数据分析，涵盖用户价值分层、商品类目分析、地域 GMV 分析与物流配送质量分析，最终输出 Power BI 交互式仪表盘。

## 📊 数据来源

- **数据集**：[Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
- **数据规模**：约 10 万条订单记录，覆盖 2016-2018 年
- **主要字段**：订单信息、用户信息、商品信息、支付信息、评分信息、物流信息

---

## 🔍 分析模块

### 1. 总览指标
- 平台总 GMV、订单数、用户数、客单价
- 月度 GMV 趋势分析

### 2. 用户价值分析（RFM 模型）
- 基于 Recency（最近购买）、Frequency（购买频次）、Monetary（消费金额）构建用户分层
- 发现 F 维度全为 1（复购率接近 0%），调整为 RM 两维度打分
- 使用 NTILE 窗口函数分档，将用户分为高价值、一般价值、低价值三层

### 3. 商品类目分析
- TOP15 品类营收排名
- 二八法则验证：TOP17 品类贡献 80% 营收
- 高销售额低评分品类识别（问题品类预警）

### 4. 地域 GMV 分析
- 各州订单量、GMV、客单价分布
- 核心州与边缘州的消费差异

### 5. 物流配送分析
- 各州平均配送时长与评分关系
- 准时率统计（实际配送日期 vs 预估配送日期）
- 配送时长与用户满意度的负相关验证

---

## 💡 核心结论

| 分析维度 | 关键发现 | 业务建议 |
|---------|---------|---------|
| 用户分析 | 平台复购率接近 0%，几乎全为一次性买家 | 建立用户留存机制，重点运营高价值用户 |
| 商品分析 | TOP17 品类贡献 80% 营收（二八法则验证）| 集中资源扶持头部品类，优化低评分高销量品类 |
| 商品分析 | 床品家纺类营收第一但评分最低 | 重点排查该品类品质与物流问题 |
| 地域分析 | SP 州 GMV 占绝对主导，约为第二名 RJ 的 2.5 倍 | 深耕核心州，制定差异化区域运营策略 |
| 物流分析 | 配送时长与评分呈负相关，偏远州配送超 20 天 | 优先改善偏远地区物流时效以提升用户满意度 |

## 🛠️ 技术栈

| 工具 | 用途 |
|------|------|
| MySQL | 数据清洗、指标计算、RFM 模型、窗口函数 |
| Power BI | 交互式仪表盘、数据可视化 |

### 涉及 SQL 技术点
- 多表 JOIN、子查询、CTE（WITH 语句）
- 窗口函数：`NTILE()`、`RANK()`、`SUM() OVER()`、`COUNT() OVER()`
- 条件聚合：`CASE WHEN`
- 日期函数：`DATEDIFF()`、`DATE_FORMAT()`
